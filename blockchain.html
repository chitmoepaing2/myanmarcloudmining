Cryptocurrency ဆိုတာဘာလဲ?

Cryptocurrency ဆိုတာ medium exchange (ၾကားခံလဲလွယ္နုိင္တဲ့)
digital-asset (ဒစ္ဂ်စ္တယ္ထုတ္ကုန္တစ္ခု) ပဲျဖစ္ပါတယ္။ 

* Digital currency ရဲ့ အဓိက အားသာခ်က္မ်ား *
- လံုျခံဳစိတ္ခ်ရတဲ့ ေငြလႊဲေျပာင္းျခင္းေတြကို ျပဳလုပ္လို့ရတယ္
- စာရင္း လိမ္လည္လို့မရတဲ့ ေငြစာရင္း မွတ္တမ္းရွိတယ္
- ထို စာရင္းကို ၾကားလူတစ္ေယာက္က ၀င္ေရာက္ ဖ်က္ဆီးလို့မရဘူး၊အတုလုပ္လို့မရဘူး။
- ကမၻာအနွံ ့သို ့ အခ်ိန္တိုအတြင္း အခ်ိန္မေရြး လႊဲေျပာင္းနိုင္တယ္။

Crypto ဆိုသည္မွာ လွ်ိဳ ့၀ွက္ ေသာ စစ္သံုး စကား၀ွက္မွဆင္းသက္လာပါတယ္။
ယခင္ စစ္သံုး လွိ်ဳ ့၀ွက္ သတင္းပို့ ေသာ နည္းစနစ္ကို တိုးျမွင့္တီထြင္ရာမွ Digital code မ်ားကို ေငြေၾကးကဲ့သို့ သတ္မွတ္လႊဲေျပာင္းနိုင္တာေၾကာင့္ Crypto + currency ဟုေခၚဆိုရျခင္းျဖစ္ပါတယ္။

2008 ခုႏွစ္ခန့္မွာ Satoshi Nakamoto ဆိုတဲ့ ကြန္ျပဴတာပညာရွင္ တစ္ေယာက္ဟာ  Bitcoin ဆိုတဲ့ internet သံုး ၾကားခံ ေငြေၾကးတစ္ခု အေၾကာင္း စာတမ္းတစ္ခု ကို  အနာဂါတ္ ေငြေၾကး၀န္ေဆာင္မွု အျဖစ္ အင္တာနက္ေပၚမွာ စတင္ တင္ျပခဲ့ပါတယ္ ။ထို စာတမ္းကို Bitcoin Whitepaper လို့ ေခၚပါတယ္။ https://bitcoin.org/bitcoin.pdf

Bitcoin ရဲ့ အနာဂါတ္ ရည္မွန္းခ်က္ ၊ လုပ္ေဆာင္နိုင္စြမ္းေတြနဲ့ ထို Bitcoin ကို  ေရးသားထားတဲ့ code ေတြကို အမ်ားျပည္သူေလ့လာနိုင္ဖို့ မိတ္ဆက္ခဲ့ပါတယ္။ 

Blockchain ဆိုတာဘာလဲ?

ထို Bitcoin digital code မ်ားကို လႊဲေျပာင္းသိမ္းဆည္းေပးတာ
ကေတာ့ Blockchain ဆိုတဲ့ နည္းပညာပဲ ျဖစ္ပါတယ္။ 

Blockchain ဆိုတာဟာ က်ေနာ္တို့ mobile banking ေတြရဲ့ ေငြလႊဲစာရင္း ပံုစံမ်ိဳးပါပဲ။ မတူညီတာကေတာ့ သာမန္ mobile ေငြလႊဲေျပာင္းျခင္းေတြကို Bank ေတြရဲ့ Data centerကေန လိုသလိုထိန္းခ်ဳပ္နိုင္ျပီး ၀င္ေရာက္ျပဳျပင္ ဖ်က္ဆီးနိုင္ပါတယ္။ ဒီလို ထိန္းခ်ဳပ္လို့ရတဲ့ နည္းပညာကို centralized chain လို ့ေခၚပါ တယ္။ 

user ေတြရဲ့ ေငြပမာဏ ကိန္းဂဏန္းမ်ားရဲ့ data ေတြကို မူလ server ထဲမွာပဲသိမ္းဆည္းထားတာေၾကာင့္ ဗဟိုထိန္းခ်ဳပ္မွု စနစ္လို့ေခၚတာ ျဖစ္ပါတယ္။

Blockchain ဆိုတာကေတာ့ data ေတြကို ဗဟို serverထဲကdatabase ထဲမွာ သိမ္းထားတာမ်ိဳး မဟုတ္ပဲအင္တာနက္ လိုင္းရွိတဲ့ ဘယ္ေနရာမွာမဆိုuser ေတြရဲ ့ data ေတြကို ျဖန့္ၾကတ္ သိမ္းဆည္း ထားတာျဖစ္လို ့ ဘယ္သူ က မွ ထိန္းခ်ဳပ္လို့ မရပါဘူး။  

user တစ္ေယာက္ျခင္းကသာလွ်င္ မိမိ digital code ေတြကို စီမံနိုင္ျပီး 
လိုသလို လႊဲေျပာင္း ေပး နိုင္ ၊ လက္ခံနိုင္မွာ ျဖစ္ပါတယ္။

user တစ္ေယာက္ လႊဲေျပာင္းခဲ့ေသာ data ဟာ အင္တာနက္ေပၚမွာ စာရင္း တစ္ခုအေနနဲ့ ေပၚလာ ပါတယ္ ။ထို စာရင္းကို ledger သို့မဟုတ္ block လို့ ေခၚဆိုပါတယ္။ထိုကဲ့သို့ user မ်ားစြာရဲ့ data ေတြကို စုေပါင္းသိမ္းဆည္းလိုက္တဲ့ အခါမွာ ေတာ့ အားလံုးကို ဆက္လိုက္တဲ့ chain လိုျဖစ္ျပီး Blockchain လို့ေခၚတာ ျဖစ္ပါ တယ္။

ထို လႊဲေျပာင္းခဲ့တဲ့ စာရင္းဇယား block မ်ားကို လူတိုင္းပြင့္လင္းျမင္သာ စြာ ၾကည့္ ရွဴ ့ နိုင္ ေအာင္ blockchain website မွာ ျပသေပးထား ပါတယ္။ဘယ္ေန့ ဘယ္အခ်ိန္မွာ ဘယ္ ပမာဏ ကို လႊဲခဲ့လည္းဆိုတာ ၾကည့္ရွဳ နိုင္ ပါတယ္။
https://blockchain.info


* Blockchain မွာ 2 ပိုင္း ရွိပါသည္ *
Block + chain (Block ဆိုတာ မွတ္တမ္းတစ္ခု ၊ chain ဆိုတာ ဆက္ေပးတာ)
သာမန္ server ေတြမွာ ေငြလႊဲေျပာင္းမွုေတြကိုအလြယ္တကူ ျပဳ ျပင္လို့ရတယ္။ Blockchain နဲ့ ျပဳလုပ္မယ္ဆိုရင္ေတာ့ေငြလႊဲ အခ်က္အလတ္ ေတြကို Block   ခြဲ တစ္ခုစီမွာသိမ္း ထားျပီး ထို block ေတြၾကား က ဆက္သြယ္မွုကို လွ်ိဳ ့၀ွက္ သေကၤတ(Hash)နဲ့ ခံထားတယ္။

ထို Hash ဟာ ေရွ့ Block ေနာက္ Block ေတြ ကို ဆက္သြယ္ေပး တယ္။ တကယ္လို့ ထို Block ေတြၾကား အခ်က္အလတ္သစ္ထပ္ထည့္ မယ္ဆိုရင္ထို Hash ေတြရဲ့ဆက္သြယ္မွ ပ်က္ဆီးျပီးBlockchain ရပ္တန့္သြားမယ္။ ဒါေၾကာင့္ လုပ္ျပီးသား transaction ကို ျပဳျပင္လို့မရဘူး။လံုျခံဳတယ္။

Bitcoinဆိုတာဘာလဲ ?

Bitcoin ဆိုတာ cryptocurrency ျဖစ္ပါတယ္။ထို Bitcoin ဆိုတာ အမည္ သတ္မွတ္ခ်က္သာ ျဖစ္ပါ တယ္။ထို Bitcoin ရဲ့ တကယ့္ ပံုစံ အမွန္မွာ ေငြလႊဲေျပာင္းသိမ္းဆည္း တဲ့ Blockchain ဆိုတာျဖစ္ပါ တယ္။ထို Blockchain ထဲမွာပဲ Bitcoin ပမာဏေတြ ေငြလႊဲ ေျပာင္းမွု ေတြ ျဖစ္ေနတာ ျဖစ္ျပီး user မ်ားဟာ
သင့္ေတာ္ ရာ wallet ေတြနဲ့ သိမ္းဆည္း လႊဲေျပာင္းၾက တာျဖစ္ပါ တယ္။

Bitcoin ေတြကို သိမ္းဆည္းတဲ့ wallet ဆိုတာဟာ blockchain စနစ္ရဲ့ အစိတ္ အပိုင္းေတြျဖစ္ပါတယ္။ ထို wallet ေတြမွာ ပိုင္ရွင္သာသိတဲ့ private key နဲ့ လႊဲေျပာင္း ရာမွာသံုးတဲ့ public key ဆိုတာရွိပါတယ္။လႊဲေျပာင္းမွုမွာ ပို့ရမယ့္လိပ္စာအျဖစ္ public key ကို သံုးရ ပါ တယ္။

Bitcoin ေတြကို သိမ္းဆည္းတဲ့ wallet ဆိုတာဟာ blockchain စနစ္ရဲ့ အစိတ္ အပိုင္းေတြျဖစ္ပါတယ္။ ထို wallet ေတြမွာ ပိုင္ရွင္သာသိတဲ့ private key နဲ့ လႊဲေျပာင္း ရာမွာသံုးတဲ့ public key ဆိုတာရွိပါတယ္။ လႊဲေျပာင္းမွုမွာ ပို့ရမယ့္လိပ္စာအျဖစ္ public key ကို သံုးရ ပါ တယ္။

ထို wallet မွ Bitcoin ေငြပို့ေၾကာင္း ပို့လိုက္တဲ့ digital signal ေလးေတြဟာ internet network ထဲ ေရာက္သြားျပီးထို signal code ေလးေတြ ကို တန္ဖိုးတကယ္ရွိမရွိစစ္ေဆး တြက္ခ်က္ မွတ္တမ္းတင္ကာ လက္ခံသူရဲ့ wallet ဆီ ပို့ေဆာင္ရပါတယ္။ဒီလို ၾကားမွာ စစ္ေဆးတြက္ခ်က္ျပီး အေျဖထုတ္ေပးတာကို mining လုပ္တယ္လို ့ ေခၚတာျဖစ္ပါတယ္။


Mining လုပ္တယ္ဆိုတာဘာလဲ?
(သို ့မဟုတ္)
cryptocurrency ကိုတူးေဖာ္ျခင္း

Mining လုပ္တယ္ဆိုတာ ေငြလႊဲ လာ တဲ့ code ေတြကို အေျဖထြက္ဖို့ တြက္ခ်က္ရတဲ့ျဖစ္စဥ္ကို ဆိုလို တာျဖစ္ပါတယ္။ပို့ခ်င္တိုင္း ပို့လို့ ရ ေနရင္ bitcoin ဆိုတာ တန္ဖိုးရွိ မလာနိုင္ပါ ဘူး။ ပို့ေဆာင္လိုက္တဲ့ သူဆီက ထြက္သြားတဲ့ digital code ပမာဏ ဟာ ပို့သူဆီမွာ အနွုတ္ ျပ သြားမွာ ျဖစ္ျပီး ထို code ေတြဟာ internet network ထဲေရာက္ သြားမွာျဖစ္ပါ တယ္။

ဒီလို ေရာက္လာတဲ့ code ေတြဟာ ထို ပို့သူ အေနနဲ့ ယခင္က ပို့ျပီးသား code ေတြဟုတ္မဟုတ္ဆိုတာကိုnetwork ထဲမွာ တြက္ခ်က္စစ္ေဆးပါတယ္။တကယ္လို့ ထိုသူဆီက ပို့ခဲ့ျပီးသား code ေတြဆိုရင္ လက္ခံ မပို့ေပးပါဘူး။ဒီလို စစ္ေဆး မွ သာ လွ်င္ Bitcoin ေတြဟာ ၂ခါ ထပ္ အပို့မခံ ရပဲ တန္ဖိုး ရွိေန မွာျဖစ္ပါတယ္။

ဒီလို network ထဲမွာ စစ္ေဆး ျပီး တန္ဖိုးရွိေၾကာင္းသတ္မွတ္ျပီး ရင္ေတာ့ ထို code ေတြကို လက္ခံသူရဲ့ wallet ထဲပို့ေပးလိုက္ျပီး ထို လႊဲပို့ျခင္း ျဖစ္စဥ္ကို block အေနနဲ့ မွတ္တမ္း တင္ပါတယ္။ ထိုကဲ့သို့ တြက္ခ်က္ေပးေသာ သူကို Miner လိုေခၚပါတယ္။

Bitcoin Miner (သို ့မဟုတ္)Miner ဆိုတာဘာလဲ?

Bitcoin miner (သို ့မဟုတ္) Miner ဆိုတာ တြက္ခ်က္ သူ ပဲ ျဖစ္ပါတယ္။
Minerကအေျဖမွန္ေအာင္တြက္နိုင္ရင္ထိုေငြလႊဲပို့ျခင္းျဖစ္စဥ္ ျပီးေျမာက္ ပါတယ္။

ဒီလို အေျဖမွန္ေအာင္ တြက္ခ်က္ ေပးနိုင္ဖို့ အတြက္ ကြန္ပ်ဴတာ ရဲ့ တြက္ခ်က္တဲ့အစိတ္အပိုင္းေတြကို အဓိက လုပ္ေပးတဲ့ Graphic card ေတြကို အသံုးျပဳရပါ တယ္။

ထို Graphic card ေတြဟာ တြက္ခ်က္ျခင္း ေတြကို ျပဳလုပ္ရတာ မို့ စြမ္းအင္ကို သံုးစြဲရပါတယ္။ ထိုကဲ့သို့ သံုးစြဲရေသာ စြမ္းအင္ စရိတ္ ကြန္ပ်ဴတာ စရိတ္ေတြ အတြက္ bitcoin စဖန္တီးစဥ္ကထဲကပင္ေငြလႊဲျခင္းမ်ားပါတဲ့ block တစ္ခုကို ေအာင္ျမင္ေအာင္ လုပ္ေဆာင္နိုင္သူအတြက္ ေပးရန္ ဆုေငြစနစ္ကိုblockchain programထဲတြင္ ထည့္သြင္း ေရးခဲ့ျပီးျဖစ္ပါတယ္။

ပံုမွန္အားျဖင့္ 10 မိနစ္ကို 1 block ျပီးေျမာက္ရျပီး ထို block တစ္ခုမွာ ေငြလႊဲျခင္းအၾကိမ္ 1800 ေလာက္ပါ၀င္တယ္။ ဆုေငြအေနနဲ့ 1block ကို 50 Bitcoin စေပးခဲ့ရာမွ 4 ႏွစ္ တစ္ခါ တစ္၀က္ေလွ်ာ့ စနစ္ျဖစ္လို့ ယခုအခါ 12.5 bitcoin ရရွိပါမယ္။

ထိုကဲ့သို့ တြက္ခ်က္သူ နည္းေနစဥ္မွာေတာ့ code တစ္ခုကို တစ္ေယာက္ျခင္း ေျဖရွင္းရေသာေၾကာင့္လြယ္ကူေနေသာ္လည္း တြက္ခ်က္သူမ်ားလာေသာအခါ code တစ္ခုကို လူ10 ေယာက္ ၀ိုင္းတြက္သလို ျဖစ္လာပါတယ္။

ထိုအခါအေျဖကို အရင္ ဦးေအာင္ရဖို့ ဒီထက္ စြမ္းအင္ေကာင္း တဲ့ ကြန္ပ်ဴတာ ေတြ သံုးစြဲလာရပါတယ္။ ဒီလို ျဖစ္လာတဲ့အခါ ပါ၀ါပိုၾကီးသူသာ ပိုရသလိုျဖစ္လာတဲ့ အတြက္ လူတိုင္း မလုပ္နိုင္သလိုျဖစ္လာပါတယ္။ ပါ၀ါ ပိုၾကီးသူေတြသာ ရတဲ့ mining ကို solo mining လို့ ေခၚ ပါတယ္။

ယေန့မွာေတာ့ လူတိုင္းလည္း ပါ၀င္နိုင္ေအာင္ ဘံု mining server တစ္ခုဆီသို့ကြန္ပ်ဴတာ တစ္ခုျခင္းဆီမွပါ၀ါမ်ားလႊဲေျပာင္း ေပးရံုနဲ့ တူးေဖာ္လို့ရတဲ့ pool mining စနစ္ကို လုပ္ေဆာင္ေန ၾကပါတယ္။ရရွိလာေသာ ဆုမ်ားကိုေတာ့ % မ်ားျဖင့္ ခြဲေ၀ယူၾကပါတယ္။
 

Miningစက္(သို ့မဟုတ္)Bitcoinတူးတဲ့စက္ဆိုတာဘာလဲ?


Mining စက္ဆိုတာ အမည္ေၾကာင့္ ဘယ္လို စက္ၾကီးလည္း ထင္ေနတာပါ။တကယ္ေတာ့ mining ဆင္တယ္ဆိုတာ ကြန္ပ်ဴတာ ဆင္တာနဲ့ တူတူပါပဲ။

ပံုမွန္အားျဖင့္ ကြန္ပ်ဴတာမွာ Desktop ရွိမယ္။ေဘးမွာ system box ၾကီးပါမယ္။
ထို system box ထဲမွာ hardware ေတြပါမယ္။ Mining မွာ အဓိက သံုးတာ ထို System Box ထဲ က Graphic card ရဲ့ တြက္ခ်က္မွု စြမ္းရည္ကိုပါ။

ထို graphic card ဆိုတာ game ဆိုင္ေတြမွာ သံုးတဲ့ card ကို ဆိုလိုတာပါ ။ဒါေပမယ့္ Graphic card တခုထဲနဲ့ မလံုေလာက္ေတာ့ Graphic card အမ်ားၾကီးကို တပ္ဆင္ဖို ့ extension လုပ္ျပီး ဆင္ရပါတယ္။ဒါေၾကာင့္ mining စက္ဆိုရင္ Graphic card ေတြဆင္ထားတာ ကို ျမင္ရတာျဖစ္ပါတယ္။

Mining လုပ္တဲ့အခါမွာ ကြန္ပ်ဴတာ လို တပ္ဆင္ျပီး တူးေဖာ္ ၾကတာ ကို သိရွိျပီး ျဖစ္တဲ့အတြက္ mining လုပ္ရာကေနရနိုင္တဲ့ အက်ိဳးအျမတ္ကို သိဖို့လိုပါတယ္။

ပံုမွန္အားျဖင့္ တူးေဖာ္သူမ်ားတဲ့အခါ ရလာတဲ့ ခြဲတမ္း နည္းတတ္ေပမယ့္မိမိ တူးေဖာ္ေနေသာ coin ရဲ့ ေစ်းနွုန္း က တိုးတက္လာ တယ္ဆိုရင္ အက်ိဳးအျမတ္ရနွုန္းက ပိုမို မ်ားျပားတတ္ပါတယ္။Graphic card တစ္ခုရဲ ့မီးသံုးအားဟာ Graphic card အမ်ိဳးအစားေပၚမူတည္ျပီး 80watt ,120watt ၀န္းက်င္ရွိပါတယ္။

Mining လုပ္ေနသူတစ္ေယာက္ရဲ့ တစ္လမီတာခဟာတူးေနတဲ ့coinအမ်ိဳးအစားႏွင့္ေစ်းအတက္အက်ေပၚမူတည္ျပီးတစ္လရတဲ့၀င္ေငြထဲက3% မွ 5% ၀န္းက်င္ေပးရပါတယ္။တစ္လကို3သိန္းဖိုးတူးနိုင္ရင္2ေသာင္းဖိုး မီတာခ ေဆာင္ရတဲ့ သေဘာျဖစ္ပါတယ္။  

Coin ေစ်းေတြတက္လာရင္ ၇လနဲ့ အရင္းေၾကတက္ျပီး ပံုမွန္ဆို တစ္နွစ္ၾကာ ပါတယ္။ တရုတ္ႏိုင္ငံမွေစ်းေပါေသာMiningစက္အေဟာင္းမ်ားႏွင့္တူးႏိုင္ရင္ ၃လနဲ ့အရင္းေႀကႏိုင္ပါသည္။သာမန္ လုပ္ငန္းေတြထက္ေတာ့သာတဲ့ အျမတ္ရနိုင္တာမို့ အက်ိဳးရွိနိုင္ပါတယ္။ နိုင္ငံေတာ္အေနနဲ့ ကြန္ပ်ဴတာ နွင့္ ဆက္စပ္ ပစၥည္းမ်ားမွ ကမၻာ့ အဆင့္ ရင္းနွီးျမဳပ္နွံမွုတစ္ခုကို ေျမေတာင္ေျမွာက္ေပးနိုင္ပါတယ္။

Bitcoinရဲ  ့စီးပြားေရးစနစ္(ecosystem)


Bitcoin ရဲ့ စီးပြားေရးလည္ပတ္ပံုစနစ္ကို ေလ့လာတဲ့အခါ
1- Mining
2- Exchanges
3- Wallets
4- Payments
5- ATM
6- Financial Services
စတာေတြပါ၀င္တာကိုေတြ ့ရပါတယ္။


(1)Mining Ecosystem
=================
Graphic card ထုတ္လုပ္ေသာ company မ်ားရဲ့ တစ္နွစ္ ေရာင္း အားဟာ 5 ဘီလီယံေလာက္ရရွိၾကပါတယ္။ အထူးသျဖင့္cryptocurrency တူးေဖာ္သူမ်ား၀ယ္ယူျခင္းေၾကာင့္ ရရွိတာျဖစ္ပါတယ္။ ဒါဟာ cryptocurrency ေတြေၾကာင့္ရတဲ့ အက်ိဳးအျမတ္ ျဖစ္ပါ တယ္။အလားတူပဲ Antminer ေရာင္းခ်ေသာ တရုတ္ company ရဲ့ တစ္ႏွစ္စာ အျမတ္ဟာ 5 ဘီလီယံ ရွိပါတယ္။ ဒီလုပ္ငန္းၾကီးေတြ လည္ပတ္မွု ေၾကာင့္ cryptocurrency ထဲ ေငြေၾကးလည္ပတ္မွုေတြ ျဖစ္ေပၚေစပါတယ္။ ဒါဟာ ecosystem ျဖစ္ပါတယ္။

(2)Exchanges Ecosystem
====================
Cryptocurrency ကို local ေငြနဲ့လဲလွယ္ေပးရာမွ ၀န္ေဆာင္ခ ရရွိေသာ လုပ္ငန္းမ်ားလည္း ကမၻာအႏွံ့ မ်ားစြာရွိေနပါတယ္။


(3)Wallet
=======
Cryptocurrency မ်ား လႊဲေျပာင္းမွုကို ၀န္ေဆာင္မွုေပးရာမွ အက်ိဳး ျမတ္ရရွိေနေသာ လုပ္ငန္းမ်ားလည္း ရွိပါတယ္။

(4)Payments
==========
Cryptocurrency ျဖင့္ ေငြေခ် ၀ယ္ယူနိုင္ေသာ payment ၀န္ေဆာင္ မွုမ်ား လည္း ရွိေနပါတယ္။

(5)ATM
======
ATM မ်ား မွ အလြယ္တကူေငြထုတ္ယူနိုင္ေအာင္ ၀န္ေဆာင္မွု ေပးရာမွလည္း အက်ိဳးျမတ္မ်ား ရရွိေနပါတယ္။


(6)Financial Services
================
ကမၻာ အနွံ ့ေငြ လႊဲ ၀န္ေဆာင္မွုမ်ားလည္း လုပ္ေဆာင္နိုင္ရာက အက်ိဳးအျမတ္မ်ား ရရွိေနၾကပါတယ္။ ထို ecosystem မ်ားရွိေနျခင္းဟာ Bitcoin ကို ၀ယ္လိုအား ၊ ေရာင္းလိုအား မ်ွတေနေစပါတယ္။

Bitcoinေစ်းကြက္စတင္လာပံု

2008 ခုႏွစ္တြင္ Satoshi Nakamoto(ကြန္ျပဴတာပညာရွင္) ဟာ Bitcoin 1သန္း ကို ပထမဆံုး Block အေနနဲ့ စတင္ မွတ္တမ္းတင္တူးေဖာ္ျပီးေနာက္ သူရဲ့ bitcoin source code ေတြကို public မွာ တင္ထားခဲ့ျပီး ေပ်ာက္ကြယ္ သြားခဲ့ပါတယ္။

Bitcoinကို စုစုေပါင္း 21 သန္း ထုတ္လုပ္မယ္လို့ program ေရးထားခဲ့တာ ျဖစ္ပါတယ္။ က်န္ရွိခဲ့ေသာ သန္း 20 ကို အျခား computer programmer မ်ားက Bitcoin ရဲ့source code မွ ယူကာ စမ္းသပ္ တူးေဖာ္ခဲ့ျပီး payment အေနနဲ့ အေပ်ာ္သေဘာ စမ္းသံုးခဲ့ၾကပါတယ္။  

2011 ခုနွစ္မွာ ပီဇာတစ္ခုကို bitcoin 10000 နဲ့ စတင္ ၀ယ္ယူ ခဲ့ပါတယ္။ ထို ့ေနာက္မွာ Bitcoin ကို internet ေပၚက computer programmer မ်ားရဲ့ ၾကားခံ ေငြတစ္ခုအေနနဲ့ အသံုးျပဳလာ ရာကေန 2013 မွာcryptocurrency stockmarket အျဖစ္ တင္ကာလူတိုင္း၀ယ္လို့ရေအာင္ျပဳလုပ္ေပးခဲ့ပါတယ္။

ထိုအခ်ိန္မွ စတင္ကာ အျခား cryptocurrency မ်ားလည္း ေပၚေပါက္လာ ျပီး ယေန့မွာ cryptocurrency ေပါင္း 1645 ေလာက္နဲ့ coinmarket ၾကီး ျဖစ္ေပၚလာတာျဖစ္ပါတယ္။

Bitcoin ကိုဘယ္လိုရယူႏိုင္မလဲ?

Bitcoin digital code ကို ရယူနိုင္တဲ့ နည္းလမ္း ၂ နည္းရွိပါတယ္။
(1) ၀ယ္ယူမယ္ (တစ္ျခားတစ္ေယာက္ထံမွျပန္လည္၀ယ္ယူျခင္းျဖစ္ပါတယ္)
(2) တူးယူမယ္ (ကိုယ္တိုင္Miningလုပ္ျခင္းျဖင့္ရယူျခင္းျဖစ္ပါတယ္)
၀ယ္ယူတယ္ဆိုတာကို ၂ ေနရာမွာ လုပ္လို့ရပါတယ္.။
(1)Miningလုပ္ေနသူေတြဆီကေန၀ယ္ယူႏိုင္ပါတယ္(offline)
(2)Exchange ေတြဆီကေန၀ယ္ယူႏိုင္ပါတယ္(online)


Exchange ေတြဆီမွာ၀ယ္ဖို့ visa,master,credit card ေတြကို သံုးျပီး ၀ယ္နိုင္ ပါတယ္။ ေနာက္ပိုင္းမွာေတာ့ credit card ကို ေပးမသံုးပဲ မိမိစာရင္းရွိေငြ visa, master card ေတြနဲ့သာ ၀ယ္ေစပါတယ္။

နိုင္ငံ တစ္ခုနဲ့ တစ္ခု bitcoin အေပၚ သေဘာထားမတူၾကသလို ထိုသေဘာ ဟာအခ်ိန္တိုင္း ေျပာင္းလဲေနပါတယ္။ ဗဟိုဘဏ္ အခ်ိဳု ရဲ့ cryptocurrency အေပၚ အျမင္ဟာ နိုင္ငံျခား သံုးေငြ ဆိုတဲ့ အျမင္ ရွိေနပါတယ္။ ဒီလို ဆို ရင္ နိုင္ငံျခားသံုးေငြ ဆိုင္ရာ ဥပေဒနဲ့ အဆင္မေျပနိုင္ ပါဘူး။ Cryptocurrency ကို လက္ခံလာတဲ့ နိုင္ငံၾကီးေတြကေတာ့ crypto ကို နိုင္ငံျခားသံုးေငြလို့ မျမင္ပါဘူး။

2 မ်ိဳး ခြဲျခားထားတာေတြ့ရပါတယ္။
(1) Digital medium exchange အေနနဲ့ျမင္တယ္(amazon ေစ်း၀ယ္codeလို)
(2) Digital asset အေနနဲ့ျမင္တယ္(အိမ္ျခံေျမ လို asset)

US လို နိုင္ငံၾကီးေတြက Digital medium exchange အေန နဲ့ေရာ crypto asset အေနနဲ့ေရာ လက္ခံၾကပါတယ္။ တခ်ိဳ ့နိုင္ငံေတြကေတာ့ commodity(asset)ထဲ သတ္မွတ္ ျပီး regulate လုပ္ၾကတယ္။

Commodity ဆိုတာ Stock exchange ေခါင္းစဥ္ေအာက္က အစုရွယ္ရာ၊ ေရႊ ၊ ေဒၚလာ ၊ ေရနံ စတာေတြျဖစ္ပါတယ္။ Crypto currency ကို Digital asset လို့ သတ္မွတ္ျပီး ခြင့္ျပဳ လိုက္တာ ျဖစ္ပါ တယ္။

စကၤာပူ မွာ ဆိုရင္ Bitcoin အတြက္ အလြယ္တကူ ေငြလဲနိုင္ ေအာင္ Bitcoin ATM 25 ခု လုပ္ေပးထားတယ္။ Exchanger ေတြကို အခြန္ ေကာက္ ယူ ၾကပါတယ္။ထို Exchanger ေတြဟာ local bank ေတြနဲ့ ခ်ိတ္ဆတ္မွု ရွိရ ပါတယ္။

ေယဘုယ်အားျဖင့္ local bank ေတြ နိုင္ငံတကာ အဆင့္မွီ ၀န္ေဆာင္မႈ ေတြ ရွိလာနိုင္မွlocal crypto exchange ေတြ တိုးတက္လာတာေတြ ့ရပါတယ္။ Cryptocurrency ဆိုတဲ့ အမည္ေၾကာင့္ currency အျဖစ္ ေရာေထြးမွုမျဖစ္ ေအာင္ Digital asset လို့သာ သံုးနွုန္းၾကတာ ကို ေတြ ့ရပါတယ္။

Cryptocurrency ကိုဘယ္လိုထိန္းခ်ဳပ္မလဲ?


Cryptocurrency ကို ထိန္းခ်ဳပ္ဖို့ စဥ္းစားရင္အားသာခ်က္ေတြကို ရနိုင္တဲ့ နည္းလမ္းဘက္ကိုပညာပိုေပးျပီး အားနည္းခ်က္ ေတြဘက္ကို ေတာ့ ထိန္းခ်ဳပ္ ရမွာျဖစ္ပါတယ္။ Cryptocurrency ကုိ နိုင္ငံျခား သံုးေငြ ဆိုတာထက္ Digital asset အေနနဲ့သာ သံုးသပ္ ဆံုးျဖတ္ရတာ ပို အျမင္ ရွင္းပါ လိမ့္္္မယ္။

Local exchanger မ်ားကို စနစ္တက် မွတ္ပံုတင္ေစျပီး local exchanger website ေတြမွာ မွတ္ပံုတင္လာမယ့္ user မ်ားရဲ့ KYC ကို မျဖစ္မေန ေတာင္း ခံရမွာျဖစ္ပါတယ္။ တစ္ေယာက္ခ်င္းစီရဲ့ ၀ယ္ယူတဲ့ပမာဏ ေရာင္းခ်တဲ့ ပမာဏ မွတ္တမ္းေတြ ရွိမွ သာလွ်င္ လိုအပ္ပါက တိုက္ဆိုင္ စစ္ေဆးနိုင္ မွာျဖစ္ပါတယ္။

Exchanger မ်ားဆီမွအခြန္ ေကာက္ယူ ရ မွာ ျဖစ္ပါတယ္။Exchanger တိုင္းမွာ user ေတြ သိရမယ့္ risk warning ေတြကို ရွင္းလင္းစြာ ျပသထားရမွာျဖစ္ပါတယ္။ သက္ဆိုင္ရာဗဟိုဘဏ္ အေနနဲ့ မိမိ ခြင့္ျပဳနိုင္တဲ့cryptocurrency ပံုစံကို ေၾကျငာေပးရမွာ ျဖစ္ပါတယ္။Digital asset အေနနဲ့သာခြင့္ျပဳနိုင္မယ္ဆိုရင္ payment အေနနဲ့ တားျမစ္ရမွာ ျဖစ္ပါတယ္။  

Cryptocurrencyေၾကာင့္ျဖစ္နိုင္တဲ့ေငြမဲခ၀ါခ်မွုကိုဘယ္လိုထိန္းခ်ဳပ္မလဲ?


Cryptocurrency ေၾကာင့္ ျဖစ္လာနိုင္တဲ့transaction ေတြမွာျပည္ပသို့ အထြက္ ျပည္ပမွ အ၀င္ ၂ မ်ိဳးသာရွိပါတယ္။အ၀င္ေရာအထြက္ေရာLocal exchanger ဆီမွာ ေငြသားေျပာင္းလဲမွု မျဖစ္မေနလုပ္ရမွာ ျဖစ္ပါတယ္။
 
ျပည္တြင္းမွာ bitcoin payment လက္ခံျခင္းမရွိတာဟာ local ေငြေၾကးကို မျဖစ္မေနလိုအပ္မွာ ျဖစ္လို့ ျပည္ပကို ပို့လွ်င္local exchanger ထံမွာ Bitcoin ကို ၀ယ္ယူဖို ့လိုျပီး ျပည္ပမွ ၀င္လာ လွ်င္လည္း local exchanger ထံမွာ ေငြသားလဲယူရမည္ျဖစ္ပါ တယ္။ဒါေၾကာင့္ KYC စနစ္ရွိေသာ Local exchanger မ်ားရွိေနဖို ့ အေရးၾကီးတာျဖစ္ပါတယ္။

Blockchain နည္းပညာ အသံုး၀င္မွုမ်ား


Blockchain နည္းပညာ ရဲ  ့အသံုး၀င္မွုကုိ နားလည္ေနမွ အသံုးျပဳ မွုကို နားလည္မွာ ျဖစ္ပါတယ္ ။Blockchain ဆိုတာ ေရွ့မွာ ရွင္းျပခဲ့သလို ပါပဲ အင္တာနက္ ထဲက computer တိုင္း မွာ ရွိေနနိုင္ပါတယ္။Bitcoin တို့လို cryptocurrency ေတြဟာ ထို blockchain နဲ့ တည္ေဆာက္ရ တဲ့ mobile,website application ေတြသာ ျဖစ္ပါတယ္။

Blockchain ကို ထပ္သံုးနိုင္တဲ့ ေနရာေတြကေတာ့
(1) ပိုင္ဆိုင္မွု စာခ်ဳပ္စာတန္းေတြ
(2) သေဘာတူညီမွု စာခ်ဳပ္ေတြ
(3) မဲ ေပး စနစ္ေတြ
စသည္တို ့မွာထပ္သုံးႏိုင္ပါတယ္။


(1) ပိုင္ဆိုင္မွု စာခ်ဳပ္
=============
ပိုင္ဆိုင္မွုစာခ်ဳပ္ေတြလုပ္တဲ့အခါမွာ စာရြက္ေတြနဲ့ လုပ္ရင္ ျပဳျပင္ ရ လြယ္တယ္ ။Blockchain နဲ့ ျပဳလုပ္မယ္ဆိုရင္ေတာ့စာခ်ဳပ္ပါ အခ်က္အလတ္ ေတြကို Block တစ္ခုစီမွာ ခြဲ သိမ္း ထားျပီးထို block ေတြၾကား က ဆက္သြယ္မွုကို လွ်ိဳ ့၀ွက္ သေကၤတ(Hash)နဲ့ ခံထားတယ္။  
 

ထို Hash ဟာ ေရွ့ Block ေနာက္ Block ေတြကို ဆက္သြယ္ ေပး တယ္။ တကယ္လို့ ထို Block ေတြၾကား အခ်က္အလတ္သစ္ ထပ္ထည့္ မယ္ဆိုရင္ ထို Hash ေတြရဲ့ ဆက္သြယ္မွ ပ်က္ဆီးျပီး ထို Blockchain ရပ္တန့္သြားမယ္။ ဒါေၾကာင့္ လုပ္ျပီးသား စာခ်ဳပ္ကို ျပဳျပင္လို့မရဘူး။လံုျခံဳတယ္။


Smart Contract
============
ပံုမွန္အားျဖင့္ အေရာင္းအ၀ယ္စာခ်ဳပ္ခ်ဳပ္တဲ့အခါမွာ ဘယ္ေန့ ဘယ္ရက္မွာ ေငြ ေပးမယ္ဆိုတာေတြကို ခ်ဴပ္ဆိုၾကတယ္။ ဒီလိုခ်ဳပ္ဆိုမွုေတြက စာရြက္ေပၚမွာပဲမို့ ပ်က္ျပယ္သြားတဲ့ အခါ တရားရံုး စရိတ္ ေတြပါထပ္ကုန္တယ္ ။ဒီလို မျဖစ္ရေအာင္ Blockchain နဲ့ ခ်ဳပ္ဆိုၾကတယ္။


Blockchain နဲ့ smart contract ေရးလိုက္မယ္ဆိုရင္ ဘယ္ေန့ ဘယ္အခ်ိန္မွာ ေငြေခ်မယ္ဆိုတဲ့ သေဘာတူညီမွု Block နဲ့ ၂ ဖက္ ရဲ့ bank account ေတြကို ခ်ိတ္ဆတ္ေပးလိုက္တယ္။ အခ်ိန္ေစ့တာနဲ့ အလိုလို ေငြလႊဲေပးမွာျဖစ္တယ္။ဒါဟာ စာခ်ဳပ္စာတန္း ေတြကုိ ပိုမို ခိုင္မာလာေစတယ္။

Online ကေန မဲေပးျခင္းကို လုပ္ေဆာင္ေနၾကေပမယ့္ မဲခိုးလို တဲ့အခါ hacker ေတြ အကူညီနဲ့ အလြယ္တကူ လုပ္လို့ရတဲ့ အ တြက္ေၾကာင့္ အဆင္မေျပပါဘူး။Blockchain စနစ္နဲ့ မဲေပးတဲ့အခါမွာေတာ့ ဒီလို မဲစာရင္းေတြကို hack လို ့မရသ လို ျပဳျပင္လို ့လည္းမရတဲ့အတြက္ Blockchain ကို မွန္ကန္တဲ့ေရြးေကာက္ပြဲေတြ မွာ သံုးနိုင္မွာျဖစ္ပါတယ္။ သိရွိရန္လိုအပ္ေသာ အခ်က္မ်ားကိုသာ ေဖာ္ျပထားျခင္းျဖစ္ပါတယ္ ။
